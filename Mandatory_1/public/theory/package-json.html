package.json

Package.json bruges til at definere projektet for npm. Man kan oprette den selv, ved at lave filen og indsætte ’{}’. Eller man kan bruge

$ npm i

For at få hjælp med at oprette den.

package.json kan indeholde ting som dependecies, scripts, devDepedencies.

{

 "dependencies": {

  "express" : "4.17.1"

 }

}
