JavaScript Functions

Eksekverende kode køres i funktioner og kan defineres så koden kan kaldes.

function eksempel(param) {

 console.log(”Hello”);

}

eksempel();

Funktionerne defineret i JS, er first-class citizens. Så de er alle sammen registreret, så de kan kaldes hvor som helst i dokumentet.

I JS kan funktioner parses som parametre, så de kan kaldes som en callback function i funktionen.

function eksempel(callBackFunction) {

 callBackFunction();

}

eksempel( function () { console.log(”callback”) });
